# Demo 1 - Flutter Watchlist App

Hey there! 👋 Welcome to **Demo 1**, a simple Flutter app that shows the items in watchlist and enables the feature re-order the watchlist items as per user choice and save selection.

## Features

- ✨ Clean and simple watchlist interface
- 📱 Cross Platform implementation, thanks to flutter.
- 🛠️ Easy to manage code using CLean Code Architecture, and state management using Flutter BLoC.

## Getting Started

Ready to dive in? Here's how to get this app running on your machine.

### Installation

1. **Clone the repo:**
   git clone https://github.com/Sagnikc663/demo_1.git
   cd demo_1

2. **Install dependencies:**
   flutter pub get

3. **Run the app:**
   flutter run

That's it! The app should launch on your connected device or emulator.

## Project Structure

Here's a quick overview of how the code is organized:
- `lib/main.dart` - The entry point of the app
- `lib/home.dart` - The main home screen
- `lib/feature/watchlist/` - Watchlist-related logic
- `lib/widgets/watchlist_item.dart` - Reusable widget for watchlist items


Built using Flutter. Happy coding!
